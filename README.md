# How to launch
1. Make sure you're running the venv
2. Navigate to the root folder of the project
4. Make sure you install all pip requirements detailed in `requirements.txt`
5. Type `flask --app main --debug run` into your console
6. Ctrl+click on the resulting link and enjoy!
